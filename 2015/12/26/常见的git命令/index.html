常用的git命令

2015.12.26 Liu Qiang  热度 ℃

本文主要介绍Git常见的命令

1.查看本地分支

$ git branch

2.查看远程分支

$ git branch -a

3.创建分支

例如我们要创建test分支，则输入如下命令：

$ git branch test

4.切换分支

例如我们要切换到test分支，则输入如下命令：

$ git checkout test
