CSS常见QA之实现水平垂直居中

2016.03.26 Liu Qiang  热度 ℃

在前端开发中，我们经常会遇到将元素居中的情况。其中，居中又分为水平居中和垂直居中。水平居中是很容易的，直接设置元素{margin:0 auto;}就可以实现。但是垂直居中就要相对复杂一些了。
下面就讨论水平垂直居中的完整实现。

步骤一：定义元素

首先，定义一个需要垂直居中的div元素，代码如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>css实现元素水平垂直居中</title>
    <style>
        .content {
            width: 300px;
            height: 300px;
            background: orange;
        }
    </style>
</head>
<body>
    <div class="content"></div>
</body>
</html>

效果如下：

步骤二：清除浏览器默认样式

不过，在这之前，我们先要设置div元素的祖元素即html和body的高度为100%(因为它们默认高度是0)，并且清除默认样式，代码如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>css实现元素水平垂直居中</title>
    <style>
    html,
    body {
        width: 100%;
        height: 100%;
        margin: 0; /*清除默认样式*/
        padding: 0; /*清除默认样式*/
    }

    .content {
        width: 300px;
        height: 300px;
        background: orange;
        margin: 0 auto; /*水平居中*/
    }
    </style>
</head>
<body>
    <div class="content"></div>
</body>
</html>

清除默认样式后，效果如下：

步骤三：实现水平居中

可以通过设置{margin: 0 auto}实现水平居中,代码如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>css实现元素水平垂直居中</title>
    <style>
        .content {
            width: 300px;
            height: 300px;
            background: orange;
            margin: 0 auto;
        }
    </style>
</head>
<body>
    <div class="content"></div>
</body>
</html>

效果如下：

通过此步骤，我们已经实现了水平居中，接下来实现比较难的垂直居中。

步骤四：实现垂直偏移

接下来，需要让div往下移动，我们知道top属性可以使得元素向下偏移，但是，由于默认情况下，元素在文档流里是从上往下、从左往右紧密布局的，我们不可以直接通过top属性改变它的垂直偏移，这就需要使用position属性，设置它的值为relative。
垂直偏移需要用到top属性，它的值可以是具体的像素，也可以是百分数。因为我们现在不知道父元素（即body）的具体高度，所以，是不可以通过具体像素来偏移的，而应该用百分数。既然是让它居中，有人想让它的值为50%不就可以了吗？问题真的就是这么简单，我们来试下设置{top:50%},会出现什么效果，代码如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>css实现元素水平垂直居中</title>
    <style>
    html,
    body {
        width: 100%;
        height: 100%;
        margin: 0; /*清除默认样式*/
        padding: 0; /*清除默认样式*/
    }

    .content {
        width: 300px;
        height: 300px;
        background: orange;
        margin: 0 auto; /*水平居中*/
        position: relative;
        top:50%;
    }
    </style>
</head>
<body>
    <div class="content"></div>
</body>
</html>

效果如下：

我们发现，此时div在垂直方向上并没有居中，明显是向下偏移了。我们在浏览器中画一条红色的线条用来参考，如图所示：

步骤五：实现垂直居中

通过步骤四，我们发现，要想让div居中，只需要让div的中心移动到红线的位置，就可以了。那么问题来了，怎么让它的中心移动到红线处呢？从图中可以观察到，从div的中心到红线的距离是div自身高度的一半。这时候，我们可以使用通过 margin-top属性来设置，因为div的自身高度是300px，需要设置他的margin-top值为-150px。为什么是要设置成负数的呢？因为正数是向下偏移，我们是希望div向上偏移，所以应该是负数，代码如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>css实现元素水平垂直居中</title>
    <style>
    html,
    body {
        width: 100%;
        height: 100%;
        margin: 0; /*清除默认样式*/
        padding: 0; /*清除默认样式*/
    }

    .content {
        width: 300px;
        height: 300px;
        background: orange;
        margin: 0 auto; /*水平居中*/
        position: relative;
        top:50%;
        margin-top: -150px;
    }
    </style>
</head>
<body>
    <div class="content"></div>
</body>
</html>

效果如下：

我们发现此时div确实居中了，well done，终于达到我们要的效果了，哈哈。

除了可以使用margin-top把div往上偏移之外，CSS3的transform属性也可以实现这个功能，通过设置div的transform：translateY(-50%)，意思是使得div向上平移自身高度的一半（50%），代码如下：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>css实现元素水平垂直居中</title>
    <style>
    html,
    body {
        width: 100%;
        height: 100%;
        margin: 0; /*清除默认样式*/
        padding: 0; /*清除默认样式*/
    }

    .content {
        width: 300px;
        height: 300px;
        background: orange;
        margin: 0 auto; /*水平居中*/
        position: relative;
        top:50%;
        transform: translateY(-50%);
    }
    </style>
</head>
<body>
    <div class="content"></div>
</body>
</html>

使用CSS3的flex弹性布局实现水平垂直居中

使用CSS3的弹性布局很简单，只要设置父元素（这里是指body）的display的值为flex即可。具体代码如下，对代码不做过多的解释，如果想了解弹性布局的可以看阮一峰老师的博客(http://www.ruanyifeng.com/blog/2015/07/flex-grammar.html)：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>index</title>
    <style>
        html,body {
            width: 100%;
            height: 100%;
            margin: 0;
            padding: 0;
        }

        body {
            display: flex;
            align-items: center; /*定义body的元素垂直居中*/
            justify-content: center; /*定义body的里的元素水平居中*/
        }
        .content {
            width: 300px;
            height: 300px;
            background: orange;        
        }
    </style>
</head>
<body>
    <div class="content"></div>
</body>
</html>
